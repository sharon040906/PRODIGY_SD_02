# 🎯 Number Guessing Game (C)

A fun console-based game where the user tries to guess a randomly generated number.

## 🚀 Features
- Random number generation
- User input handling
- Feedback system (Too high / Too low)
- Multiple attempts until correct guess

## 🛠️ Technologies Used
- C Programming

## 📌 How to Run
1. Compile the program:
   gcc guessing_game.c -o game
2. Run:
   ./game

## 🎮 Gameplay
- The system generates a random number
- User guesses the number
- Program gives hints until correct

## 💡 Learning Outcomes
- Loops and conditions
- Random number generation
- Game logic implementation
## 🧠 Key Concepts Explained

### 🔹 Random Number Generation
The program uses functions like `rand()` to generate unpredictable numbers, making the game dynamic.

### 🔹 Loops
Loops (such as `while`) allow the program to repeat actions until a condition is met.

### 🔹 Conditional Statements
Conditions (`if`, `else`) help in decision-making, such as checking whether the guessed number is too high or too low.

### 🔹 User Input
The program takes input from the user during runtime to make the game interactive.
